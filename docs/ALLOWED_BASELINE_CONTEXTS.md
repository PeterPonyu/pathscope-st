# Brand independence and baseline-use boundary

Verification date: 2026-06-07

These project folders are independent local research tracks. Third-party repositories and paper names may appear only in provenance, citation, benchmark, and adapter documentation. They must not be used as local package names, API names, figure branding, or manuscript claims.

Allowed uses:

1. cite the reference paper;
2. run the frozen baseline code for parity or ablation;
3. write thin adapters that are explicitly named as baseline adapters;
4. compare metrics, runtime, inputs, and outputs under reproducible conditions.

Forbidden uses until a separate license/design review passes:

1. copying upstream source into local `src/`;
2. reusing upstream class/function names as project branding;
3. restating upstream abstracts, figure captions, or claims as local novelty;
4. implying biological/clinical performance before local evidence exists;
5. mixing article licenses with software repository licenses.

All local contribution language should emphasize the refinement/enhancement listed in the README and CLAIM_LEDGER, not the baseline name.
